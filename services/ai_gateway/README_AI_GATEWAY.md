## Структура

```text
ai_gateway/
├── __init__.py    # пакет AI Gateway
├── app.py         # FastAPI-интерфейс AI-сервиса
├── config.py      # настройки моделей, провайдеров и RAG
├── context.py     # системный промпт и оперативный контекст
├── guard.py       # защита от циклов, спама и лишних вызовов
├── memory.py      # память диалога и извлечение сущностей
├── orchestrator.py # оркестрация LLM, fallback и streaming
├── prompt.py      # правила ответов для каналов
├── providers.py   # клиенты LLM-провайдеров
└── rag_client.py  # клиент сервиса базы знаний RAG
```
