## Структура

```text
rag_service/
├── __init__.py    # пакет RAG-сервиса
├── app.py         # FastAPI API поиска и документов
├── chunking.py    # разбиение документов на фрагменты
├── config.py      # настройки RAG-сервиса
├── embeddings.py  # создание векторных представлений
├── http_pool.py   # общий HTTP-клиент
├── parsing.py     # разбор PDF, DOCX, HTML и текста
├── pipeline.py    # загрузка документов и поиск
├── rerank.py      # дополнительное ранжирование результатов
└── store.py       # Qdrant и метаданные документов в PostgreSQL
```
