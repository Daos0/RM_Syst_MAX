## Структура

```text
leads_service/
├── __init__.py           # пакет сервиса заявок
├── app.py                # FastAPI-эндпоинты приёма лидов
├── bitrix_forwarder.py   # передача заявок в Bitrix24
├── client.py             # внутренний клиент сервиса
├── contacts.py           # объединение контактных данных
├── forwarder.py          # уведомления о лидах в Telegram
├── parser.py             # извлечение сущностей из текста
├── schemas.py            # схемы запросов и ответов
├── service.py             # основная логика обработки заявок
└── storage.py            # сохранение и чтение лидов
```
