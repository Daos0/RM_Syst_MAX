## Структура

```text
shared/
├── __init__.py       # общий пакет проекта
├── config.py         # общие настройки и права доступа
├── rag_profiles.py   # создание и выбор активного профиля RAG
├── ttl_cache.py      # кэш настроек с ограниченным временем жизни
├── ai/               # клиент AI Gateway
├── broadcast/        # очередь рассылок
├── db/               # модели, сессии и репозитории БД
├── domain/           # доменные настройки AI и RAG
├── leads/            # общая логика контактов и лидов
├── prompts/          # хранилище системных промптов
└── schedule/         # расписания AI и менеджеров
```
