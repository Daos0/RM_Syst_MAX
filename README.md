# RM Systems — боты и мини-приложения для Telegram и MAX

Исходный код рабочего цифрового сервиса с ботами и единым Mini App для **Telegram** и **MAX**. Пользователи создают личные, семейные и совместные списки покупок прямо в привычном мессенджере.

## Открыть ботов

- **Telegram:** [открыть @KupiSpisok_Bot](https://t.me/KupiSpisok_Bot)
- **MAX:** [открыть бота и Mini App](https://max.ru/se13507757_bot)

## Возможности

- личные, семейные и совместные списки;
- общий каталог товаров, блюд и рецептов;
- поиск и добавление собственных позиций;
- приглашения по ссылке и совместные покупки;
- обновление открытых списков в реальном времени;
- управление списками через бота или Mini App;
- единая бизнес-логика и PostgreSQL для Telegram и MAX.

## Где находится код

```text
MAX/
├── app/platforms/max/        # бот и авторизация MAX
├── app/platforms/telegram/   # Telegram-бот и авторизация
├── app/bots/                 # общие сценарии ботов
├── app/static/               # общее Mini App
├── app/api/                  # FastAPI и бизнес-операции
├── app/db/                   # модели данных и каталог
├── migrations/               # миграции PostgreSQL
└── tests/                    # тесты проекта
```

Большая структура репозитория сохранена как пример расширения проекта: её можно дополнять новыми платформами, API, AI-модулями, CRM-интеграциями и цифровыми сервисами для бизнеса.

Подробная техническая документация находится в [`MAX/README.md`](MAX/README.md).
